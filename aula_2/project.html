Um breve resumo da aula 01

Aprendemos a como começar, a primerira liha precisa ser o "DOCTYPE html".

Na proxima linha "head"=inicio e "/head"=sendo o final, dando enter entre eles.

Na proxima linha seria o "body"=inicio e o "/body"=final, dando enter entre eles.

Na ultima linha seria "/html".

Tem a tag: "p", "br". O "p" seria para linha e o "br" é para quebra de linha.

tag: "ol", "ul", "il". O "ol" é para criar uma lista, o "ul" seria para uma lista não ordenada e o "il" para incerrir elementos na lista.

tag: "img" é para colocar um imagem a sua escolha.

tag: "href", "iframe". O "href" coloca links para acesso e o "iframe" coloca vídeos por exemplo do youtube.

tag: "table" é para fazer uma tabela.
